## 🛠 Изученные темы модуля:

### 📑 Основы и чтение
*   **3.1 - 3.2** Понимание типов файлов и форматов.
*   **3.3 - 3.4** Все варианты чтения данных (`.read()`, `.readline()`, `.readlines()`).
*   **3.5** Эффективная итерация по файлу (построчное чтение).

### ⚙️ Тонкая настройка
*   **3.6** Работа с кодировками и байтами (понимание `utf-8`).
*   **3.7** **Контекстный менеджер** — использование оператора `with` для безопасной работы.

### ✍️ Запись и режимы
*   **3.8** Запись данных в файл (`write`).
*   **3.10** Добавление информации в конец файла (режим `a`).
*   **3.11** Режимы открытия бинарных файлов (`rb`, `wb`).
